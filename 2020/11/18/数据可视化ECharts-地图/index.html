数据可视化ECharts-地图

本文最后更新于：5 个月前

地图图表的使用方式

百度地图api

需要申请百度地图ak

矢量地图

需要准备矢量地图数据

矢量地图的实现步骤

1）ECharts最基本的步骤：引入ECharts，初始化dom，初始化容器，设置option

2）准备中国的矢量地图json文件，放到json/map/的目录下面china.json

3）使用ajax获取china.json

4）在回调函数中往echarts全局对象注册地图的json地图数据

5）在geo下面设置，type：‘map’ map：’chinaMap’

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <script src = lib/echarts.min.js></script>
    <script src="lib/jquery.min.js"></script>
</head>
<body>
<div style="width: 600px;height:400px"></div>
<script>

    var mCharts = echarts.init(document.querySelector("div"))
    $.get('json/map/china.json',function (ret) {
        echarts.registerMap('chinaMap',ret)
        var option = {
            geo:{
                type:'map',
                map:'chinaMap'
            }
        }
        mCharts.setOption(option)
    })

</script>
</body>

</html>

常用配置

拖动缩放

可以用鼠标滑轮放大缩小

或者拖动进行位置移动

roam:true

显示名称

label:{
    show:true 
}

初始化缩放比例

zoom:1 //初始化缩放比例

地图中心点

center:[87.61,43.79]

显示某一个区域

1）加载该区域的矢量地图数据

2）通过registerMap注册到echarts全局对象中

3）指明geo下面的type和map属性

4）通过zoom放大该区域

5）通过center定位中心店

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <script src = lib/echarts.min.js></script>
    <script src="lib/jquery.min.js"></script>
</head>
<body>
<div style="width: 600px;height:400px"></div>
<script>

    var mCharts = echarts.init(document.querySelector("div"))
    $.get('json/map/anhui.json',function (ret) {
        echarts.registerMap('anhui',ret)
        var option = {
            geo:{
                type:'map',
                map:'anhui',
                roam:true,
                label:{
                    show:true
                }
            }
        }
        mCharts.setOption(option)
    })

</script>
</body>

</html>
