数据可视化ECharts-高级使用动画相关

本文最后更新于：3 个月前

加载动画

在ECharts获取数据中，数据没有呈现出来的时候，ECharts提供的一个更友好的效果

ECharts已经内置好了加载数据的动画，我们只需要在合适的时机显示或者隐藏即可

显示/隐藏加载动画

mCharts.showLoading() // 在获取数据之前, 显示加载动画
$.get('data/test_data.json', function (ret) {
    mCharts.hideLoading() // 当服务器数据获取成功之后, 隐藏加载动画

增量动画

数据又发生了变化，那么就会出现增量动画

var btnModify = document.querySelector('#modify')
btnModify.onclick = function () {
    var newYDataArr = [68, 32, 99, 77, 94, 80, 72, 86]
    // setOption 可以设置多次
    // 新的option 和 旧的option
    // 新旧option的关系并不是相互覆盖的关系, 是相互整合的关系
    // 我们在设置新的option的时候, 只需要考虑到变化的部分就可以
    var option = {
        series: [
            {
                data: newYDataArr
            }
        ]
    }
    mCharts.setOption(option)
}

option可以设置很多回

新的option和旧的option并不是相互覆盖的关系，而是整合的关系

设置的时候，只需要考虑变化的部分就可以了

echarts会找到两组数据之间的差异然后通过合适的动画去表现数据的变化

动画常用配置

开启动画

animation：true

动画时长

animationDuration：5000

缓慢动画

animationEasing：‘bounceOut’

动画阈值

animationThreshold：8

大于这个这个阈值会关闭动画

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>

    <script src="lib/echarts.min.js"></script>
    <script src="lib/jquery.min.js"></script>
</head>
<body>
<div style="width: 600px;height:400px"></div>
<script>
    var mCharts = echarts.init(document.querySelector("div"))
    var xDataArr = ['张三', '李四', '王五', '闰土', '小明', '茅台', '二妞', '大强']
    var yDataArr = [88, 92, 63, 77, 94, 80, 72, 86]
    var option = {
        animation: true,  // 控制动画是否开启
        // animationDuration: 7000, // 动画的时长, 它是以毫秒为单位
        animationDuration: function(arg){
            console.log(arg)
            return 2000 * arg
        },
        animationEasing: 'bounceOut', // 缓动动画
        animationThreshold: 7, // 动画元素的阈值
        xAxis: {
            type: 'category',
            data: xDataArr
        },
        yAxis: {
            type: 'value'
        },
        series: [
            {
                type: 'bar',
                data: yDataArr,
                markPoint: {
                    data: [
                        {
                            type: 'max', name: '最大值'
                        },
                        {
                            type: 'min', name: '最小值'
                        }
                    ]
                },
                markLine: {
                    data: [
                        {
                            type: 'average', name: '平均值'
                        }
                    ]
                },
                label: {
                    show: true,
                    rotate: 60
                },
                barWidth: '30%'
            }
        ]
    };
    mCharts.setOption(option)
</script>
</body>


</html>
