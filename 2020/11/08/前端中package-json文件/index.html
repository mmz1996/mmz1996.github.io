结构解析

name 项目名称

version 版本号

description 项目描述

author 作者

private 为true，代表npm拒绝发布它，私有存储库

scripts 指定了运行一些脚本命令，npm run dev

dependencies 整个项目正常运行的时候依赖的包，也就是用户在npm install的时候，安装都是自动安装的这些依赖

devDependencies 项目开发时候依赖度的工具包，运行时不依赖于这些包

engines 指定项目的工作环境，除非用户自己设置，这个字段只是建议值

browserslist 在不同的前端工具之间共用目标浏览器和node版本的配置工具
