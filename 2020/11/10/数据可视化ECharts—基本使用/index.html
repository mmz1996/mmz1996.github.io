本文最后更新于：20 天前

ECharts.js的介绍

是一个javascript实现的开源可视化库，兼容性强，底层依赖矢量图ZRender，提供直观，交互丰富，可高度个性化定制的数据可视化图表。

开源免费/功能丰富/社区活跃

ECharts.js的特点

丰富的可视化类型

折线图，柱状图，饼图，K线图

多种数据支持

key-value

二维表

TypedArray

流数据支持

流数据的动态渲染

增量渲染技术

ECharts.js上手

基本使用

步骤1：引入echart.js

步骤2：准备一个呈现图表的盒子 div

步骤3：初始化echarts实例对象

步骤4：准备配置项

步骤5：将配置项设置给echarts实例对象

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
<!--    步骤1：引入echart.js-->
    <script src="lib/echarts.min.js"></script>

</head>
<body>
<!--步骤2：准备一个呈现图表的盒子 div-->
    <div style="width: 600px;height: 400px">

    </div>
    <script>
    // 步骤3：初始化echarts实例对象
    //参数，dom决定图表最终呈现的位置
    var mCharts =  echarts.init(document.querySelector('div'))
    // 步骤4：准备配置项
        var option = {
            xAxis:{
                type:'category',
                data:['小明','小红','小王']
            },
            yAxis:{
                type:'value'
            },
            series:[
                {
                    name:'语文',
                    type:'bar',
                    data:[70,92,87]
                }
            ]
        }
        //步骤5：将配置项设置给echarts实例对象
        mCharts.setOption(option)
    </script>
</body>
</html>

ECharts的基本使用

xAxis：直角坐标系中的x轴

yAxis：直角坐标系中的y轴

series：系列列表，每个系列通过type决定自己的图表类型

柱状图

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <script src="lib/echarts.min.js"></script>
</head>
<body>
    <div style="width: 600px;height: 400px"></div>
    <script>
        var chars = echarts.init(document.querySelector("div"))
        var xData = ['张三','李四','王五']
        var yData = [88,92,63]
        var options = {
            xAxis:{
              type:'category',
              data:xData
            },
            yAxis: {
              type:'value'
            },
            series:[
                {
                    name:'语文',
                    type:'bar',
                    data:yData
                }
            ]
        }
        chars.setOption(options)
    </script>
</body>
</html>
